<objective>
Fix bulk import partial success by removing FormRequest-level geometry type validation that blocks mixed valid/invalid feature payloads from reaching the service layer.

Purpose: The current implementation validates geometry type at the FormRequest level (line 21: `'features.*.geometry.type' => ['required', 'in:Polygon']`), which causes Laravel to return a 422 validation error before the request reaches the controller when any feature has a non-Polygon geometry. This prevents the partial success pattern where valid features should be imported and invalid features should return error messages.

The ParcelService::importGeoJsonFeatures method already implements per-feature validation with try-catch error handling, but it never gets called when FormRequest validation fails. The GeoJsonPolygon rule used in the service layer provides proper geometry validation with detailed error messages.

Output: Bulk import endpoint accepts mixed valid/invalid payloads and returns 200 with imported count and errors array.
</objective>

Service layer validation (already implemented, will handle geometry validation):
```php
private function validateGeoJsonFeature(array $feature): void
{
    if (! isset($feature['geometry']) || ! is_array($feature['geometry'])) {
        throw new \InvalidArgumentException('Feature missing geometry');
    }

    // Use the shared GeoJsonPolygon validation logic
    $error = GeoJsonPolygon::validateGeometry($feature['geometry']);

    if ($error !== null) {
        throw new \InvalidArgumentException($error);
    }
}

public function importGeoJsonFeatures(array $geojsonData): array
{
    $imported = 0;
    $errors = [];

    foreach ($geojsonData['features'] as $index => $feature) {
        try {
            $this->validateGeoJsonFeature($feature);  // <-- This handles geometry validation
            // ... create parcel
            $imported++;
        } catch (\Exception $e) {
            $errors[] = [
                'feature_index' => $index,
                'error' => $e->getMessage(),
            ];
        }
    }

    return [
        'imported' => $imported,
        'errors' => $errors,
    ];
}
}
```

public static function validateGeometry(mixed $geometry): ?string
{
    if (! is_array($geometry)) {
        return 'The geometry must be a valid GeoJSON Polygon object.';
    }

    if (! isset($geometry['type']) || $geometry['type'] !== 'Polygon') {
        return 'The geometry must be of type Polygon.';
